Nara-Bot
========

An Arduino controlled Robot.

These sketches are designed for an Arduino Mega 2560/Arduino Nano.

Arduino Libraries: 
-Adafruit motorshield
-qik motor shield
-NewPing
-Servo.h

Processing: 
-Procontroll

Sketches have been tested on Windows 7 x64, and OS X 10.7.3+. As always though, YMMV.
